title: Definitions of Terms
lastmod: 2022-01-12 05:39:36 -05:00
---
			
· LIDAR: stands for Light Detection and Ranging 
· GIS: Geographic Information System 
· ESRI, ArcGIS, ArcMap, QGIS: GIS programs 
· CAD: Computer Aided Drafting 
· AutoCAD, MicroStation: CAD programs 
· Raster: Computer image file or bitmap 
· TIF, GeoTIF, IMG, JPG: Raster file formats 
· DEM or DTM: Ground model in raster format 
· Vector or Vector Layer: Data made up of graphic features 
· GeoJSON, SHP, FGDB, DXF, DWG: Vector file formats 
· Survey TXT Files: Text files containing X,Y,Z info –comma separated 
· UAV and Drone: Unmanned Aerial Vehicle · Scanner: Tripod or mobile LIDAR capture device (3D Mapping) 
· XML Layer: Z-Tools Boundary Layer 
· LAS or LAZ File: Point cloud files
